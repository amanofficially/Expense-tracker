# 💳 Expense Tracker — React + Vite + Tailwind CSS

A fully functional Expense Tracker built with React, Vite, and Tailwind CSS during my MERN Stack Internship at SVVV, Indore.

## ✨ Features
- Add income and expense transactions
- Real-time Balance, Income, Expense summary
- Category selection with emoji icons
- Delete transactions (hover to reveal)
- Green for income, Red for expense
- Scrollable transaction history

## 🛠️ Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript ES6+

## 🧠 React Concepts Used
- useState — transactions array and form fields
- useMemo — computing balance, income, expense totals
- .filter() and .reduce() — array operations
- Conditional Tailwind classes — green/red for income/expense
- Props — passing data and handlers to components
